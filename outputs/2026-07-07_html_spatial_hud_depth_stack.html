Spatial HUD Depth Stack

XR レイヤー構成を直接実装する前段として、ポインタを疑似 head pose に見立て、前景 HUD と奥行き面を分離して観察する最小プロトタイプです。ドラッグすると視差が強まります。

検証: HUD / 中景 / 奥行き面の読みやすさ 操作: pointer move + drag 差分: drag 中だけ視差を強調

Foreground HUD

読ませたい情報は手前へ。シーン内メッシュとは別の層として扱うと、XR 的な読みやすさの判断を先に試せます。

Mid Layer Prompt

中央層には状況説明や行動誘導。奥行き差を使って UI の優先順位を作ると、固定 HUD よりも空間の文脈に寄せやすいです。

Anchor Trace

低い頻度で変わる説明面はやや奥へ。将来的には gaze, hand ray, hit test 結果に応じてこの層を差し替える想定です。
