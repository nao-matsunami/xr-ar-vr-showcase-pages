Hand Joint Proximity Panel

WebXR Hand Input の 25 joint を疑似化し、index-tip と thumb-tip の pinch 距離、指先と空間パネルの近接状態を観察する最小サンプルです。

実 WebXR では `hand-tracking` を要求し、`XRInputSource.hand` の `XRJointSpace` から frame ごとに `getJointPose()` を読みます。このサンプルは同じ観察点を 2D で再現します。

joints25
pinch42mm
targetnear
radius10mm
drag: move hand / sliders: joint pose / buttons: preset hand states
